# Student-Teacher Booking Appointment System

A simple web-based Student-Teacher Appointment Booking System developed using HTML, CSS, JavaScript, and Firebase.

---

## 📌 Project Description

This project allows students to:

- Register and Login
- Book appointments with teachers
- Send purpose/message for appointment
- Store appointment data in Firebase Firestore

This project is developed as a college-level academic project.

## 🚀 How To Run This Project

1. Clone the repository:
git clone https://github.com/Hanish2006/student-teacher-booking.git

2. Open the project folder in VS Code

3. Install Live Server extension

4. Right click on `indexteacher.html`

5. Click **Open with Live Server**

6. Make sure Firebase configuration is added in `firebasetcher.js`
